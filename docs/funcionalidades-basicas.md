# Funcionalidades Básicas

## Barra de Ferramentas

- <IIcon icon="mdi:plus" height="24" />**Adicionar**: abre formulário para inclusão de novos itens nos cadastros do sistema.

- <IIcon icon="ic:baseline-view-week" height="24" /> **Colunas**: permite escolher quais colunas da tabela exibir ou ocultar.

- <span style={{verticalAlign: `middle`}}>![](/img/excel.svg)</span> **Exportar**: realiza o download de uma planilha com os dados da tabela em exibição.

## Coluna de Ações

- <IIcon icon="ic:baseline-edit" height="24" /> (**Editar**): abre formulário para edição de itens previamente cadastrados.
- <IIcon icon="ic:baseline-find-in-page" height="24" /> (**Visualizar**): abre formulário para visualização de itens em modo somente leitura.
- <IIcon icon="ic:baseline-preview" height="24" /> (**Pré-Visualizar**): abre janela para visualização rápida de documentos anexados.
- <IIcon icon="ic:baseline-cloud-download" height="24" /> (**Baixar**): efetua download de docuemnto anexo para dispositivo do usuário.
- <span style={{color:`firebrick`}}><IIcon icon="ic:baseline-delete-forever" height="24" /></span> (<span style={{color:`firebrick`}}>**Excluir**</span>): exclui um item cadastrado definitivamente. <span style={{color:`firebrick`}}>**Utilize com cuidado!**</span>
