# Medição

Uma **Medição** serve para detalhar e demonstrar os avanços físico e financeiro do contrato.

:::info[Observação]

Antes de iniciar o preenchimento do formulário de medição, é essencial que os dados do contrato estejam devidamente preenchidos em [<IIcon icon="ic:baseline-local-atm" height="24" />**Configurações de Contrato**](/mapa-sistema/configuracoes-contrato).

:::

Para acessar a aba <IIcon icon="ic:baseline-price-change" height="24" />**Medição**, abra as [**Medições do Contrato**](/mapa-sistema/contrato/medicoes) e clique no botão <IIcon icon="ic:baseline-edit" height="24" /> na coluna _Ações_.
