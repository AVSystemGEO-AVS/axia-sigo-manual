# Dados do RDO

Antes de poder utilizar todas as funcionalidades do RDO, é necessário completar informações básicas dos **Dados do RDO**.

Caso tenha ocorrido algum afastamento no dia, atualize o campo _Nº Afastamentos_.

## Jornada de Trabalho

Os horários de trabalho já estão pré-definidos para cada [**Obra**](/mapa-sistema/configuracoes-contrato/obras) e dia da semana, mas podem ser editados​, se necessário.

## Condições Climáticas por Município

Selecione um município, preencha as informações de _Tempo_, _Condição_ e _Vento_ e selecione **Incluir**. Em seguida, selecione o próximo município, se houver​. Depois de concluído, clique em **Salvar**.

:::info[Observação]

É obrigatória a inclusão das condições climáticas de pelo menos um munícipio.

:::
