# Anexos de Documentos

Os **Anexos de Documentos do RDO** são um repositório destinado a anexar documentos que sejam relevantes ao RDO.​ Podem ser incluídos tanto por esta tela quanto pelo [**Avanço Físico**](avanco-fisico#preencher-informa%C3%A7%C3%B5es-de-avan%C3%A7o).
