# RDO

O **RDO** serve para coletar todas as informações da realização das [**Obras**](/mapa-sistema/configuracoes-contrato/obras).

:::info[Observação]

Antes de iniciar o preenchimento do RDO, é essencial que os dados do contrato estejam devidamente preenchidos em [<IIcon icon="ic:baseline-local-atm" height="24" />**Configurações de Contrato**](/mapa-sistema/configuracoes-contrato).

:::

Para acessar a aba <IIcon icon="ic:baseline-summarize" height="24" />**RDO**, abra os [**RDOs do Contrato**](/mapa-sistema/contrato/rdos) e clique nos botões <IIcon icon="ic:baseline-edit" height="24" /> ou <IIcon icon="ic:baseline-find-in-page" height="24" /> na coluna _Ações_.
