# Aprovação de Boletins

Esta tela apresenta uma listagem dos **Boletins de Medição** que aguardam aprovação por parte do usuário **Aprovador**.

## Aprovar/Reprovar Boletim

Ao clicar no botão <IIcon icon="ic:check-box" height="24" />, o sistema apresentará o form com informações do boletim de medição selecionado e a opção para aprová-lo. Caso haja algum problema no documento, marque como **Reprovado** e informe o motivo para que o **Elaborador** possa resolvê-lo.

## Editar Aprovadores do Boletim

Caso algum dos aprovadores selecionados anteriormente esteja impedido de realizar a análise do boletim, é possível substitui-lo clicando no botão <IIcon icon="mdi:account-cog" height="24" />.

:::info[Observação]

Não será permitido alterar um usuário que já tenha aprovado o documento.

:::
