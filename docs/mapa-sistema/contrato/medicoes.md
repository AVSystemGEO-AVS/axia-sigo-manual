# Medições

Para adicionar uma **Medição** é pré-requisito mínimo que estejam cadastrados [**Itens de Preço e Medição**](/mapa-sistema/configuracoes-contrato/itens-preco-medicao) no contrato.

Além disso, nos casos em que a medição se dará por avanço físico proveniente do RDO ou da Coleta Física, também é obrigatório que a [**EAP Física**](/mapa-sistema/configuracoes-contrato/eap-fisica) tenha sido cadastrada e correlacionada com os itens de medição​.

Ao criar ou editar uma medição, o usuário é automaticamente redirecionado para a aba [<IIcon icon="ic:baseline-price-change" height="24" />**Medição**](/mapa-sistema/medicao).
