# Contrato

Para acessar um <IIcon icon="ic:baseline-request-quote" height="24" />**Contrato**, abra uma aba <IIcon icon="ic:baseline-corporate-fare" height="24" />**Grupo** e localize o nome do empreendimento no qual o contrato foi criado na lista à esquerda.

Abra o empreendimento clicando no botão <IIcon icon="ic:baseline-chevron-right" height="24" /> e clique duas vezes no nome do contrato.
