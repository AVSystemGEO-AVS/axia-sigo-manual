# Aprovação de RDOs

Esta tela apresenta uma listagem dos RDOs que aguardam aprovação por parte do usuário **Aprovador**.

## Aprovar/Reprovar RDO

Ao clicar no botão <IIcon icon="ic:check-box" height="24" />, o sistema apresentará o form com informações do RDO selecionado e a opção para aprová-lo. Caso haja algum problema no documento, marque como **Não Aprovado** e informe o motivo para que o **Elaborador** possa resolvê-lo.
