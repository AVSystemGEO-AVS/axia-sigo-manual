# Home

Ao logar no Sigo, o usuário é direcionado à aba <IIcon icon="mdi:home" height="24" />**Home**, na qual são apresentados os quatro **Grupos**, i.e., concessionárias Eletrobras (Eletrobras, CHESF, Eletronorte e Eletrosul) que agruparão contratos e projetos dentro do sistema.

:::warning[Importante]

Se algum dos botões estiver semitransparente, significa que o usuário não tem permissão para acessar empreendimentos e/ou contratos daquela concessionária.

Caso seja necessário acessar um grupo atualmente indisponível, entre em contato com o seu gestor e solicite a liberação do acesso.

:::

Ao selecionar um grupo, o usuário será direcionado para a aba [<IIcon icon="ic:baseline-corporate-fare" height="24" />**Grupo**](/mapa-sistema/grupo) correspondente.
