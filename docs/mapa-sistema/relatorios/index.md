# Relatórios

Para acessar os **Relatórios**, clique no <IIcon icon="mdi:menu" height="24" /> **Menu do Usuário** no canto superior direito do cabeçalho (ao lado do nome e hora) e escolha **Relatórios** para acessar.

:::warning[Importante]

Se o menu não apresentar a opção **Relatórios**, significa que o usuário não possui permissão para acessar essa seção.

:::
