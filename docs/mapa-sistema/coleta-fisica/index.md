# Coleta Física

A **Coleta Física** possui finalidade similar ao [**RDO**](/mapa-sistema/rdo), porém seu preenchimento é focado no avanço físico de atividades e coleta de fotos e anexos.​

:::info[Observação]

Antes de iniciar o preenchimento da coleta física, é essencial que os dados do contrato estejam devidamente preenchidos em [<IIcon icon="ic:baseline-local-atm" height="24" />**Configurações de Contrato**](/mapa-sistema/configuracoes-contrato).

:::

Para acessar a aba <IIcon icon="ic:baseline-note-alt" height="24" />**Coleta Física**, abra as [**Coleta Físicas do Contrato**](/mapa-sistema/contrato/coletas-fisicas) e clique no botão <IIcon icon="ic:baseline-edit" height="24" /> na coluna _Ações_.
