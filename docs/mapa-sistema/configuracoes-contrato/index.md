# Configurações de Contrato

Para acessar as <IIcon icon="ic:baseline-local-atm" height="24" />**Configurações de Contrato**, abra uma aba <IIcon icon="ic:baseline-corporate-fare" height="24" />**Grupo** e localize o nome do empreendimento no qual o contrato foi criado na lista à esquerda.

Abra o empreendimento clicando no botão <IIcon icon="ic:baseline-chevron-right" height="24" />, passe o mouse sobre o nome do contrato e clique no botão <IIcon icon="mdi:settings" height="24" />.

:::tip[Dica]

Recomenda-se preencher os cadastros na ordem apresentada no menu à esquerda (de cima para baixo), pois itens criados em um cadastro acima poderão ser necessários no preenchimento dos cadastros abaixo.

:::
