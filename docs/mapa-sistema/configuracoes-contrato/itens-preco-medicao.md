# Itens de Preço e Medição

O cadastro de **Itens de Preço e Medição** será responsável pelo controle e detalhamento de todos os valores contratuais previstos.​

Na coluna _Valor Total_, é possível acompanhar o somatório de cada item de acordo com os seus subitens, calculado automaticamente pelo sistema.

## Preenchimento manual

Para criar um novo item, clique no botão <IIcon icon="mdi:plus" height="24" />**Adicionar** na barra de ferramentas acima da tabela.

:::tip[Dica]

Na coluna _Ações_ da tabela, há outras opções para criar itens de preço:

- Utilize o botão <IIcon icon="ic:baseline-file-copy" height="24" /> para **Duplicar**, i.e., criar um novo item no mesmo nível com uma cópia dos dados do item escolhido.
- Utilize o botão <IIcon icon="ic:baseline-add-circle" height="24" /> para **Adicionar Subitem**, i.e., criar um novo item dentro do item escolhido.

:::

Insira os campos obrigatórios para criar o novo item de preço, incluindo o campo _Código_, separando os níveis por pontos. A numeração correta é essencial para garantir o posicionamento do item na hierarquia de itens de preço e medição. O campo _Nível_ será preenchido automaticamente, sendo 0 o nível raiz.

Caso o item de preço possua subitens de medição, utilize a seção correspondente para cadastrá-los.

### Itens de Medição

Preencha os campos obrigatórios para criar um novo item de medição e clique em **Incluir**. O novo item será adicionado à tabela.

Clique no botão <IIcon icon="ic:baseline-edit" height="24" /> para complementar o item com informações mais detalhadas.

:::warning[Importante]

Os dados dos itens de medição só serão efetivamente armazenados após clique no botão **Salvar**.

:::

## Preenchimento via importação de template

Alternativamente, como forma de agilizar a inserção de itens de preço e medição, pode-se fazer o download de um template para ser preenchido em aplicativos de planilha eletrônica (ex.: Microsoft Excel) clicando no botão <IIcon icon="ic:baseline-cloud-download" height="24" />**Template**.

Depois que o template for preenchido, basta importá-lo de volta para o Sigo clicando em <IIcon icon="ic:baseline-cloud-upload" height="24" />**Importar**. O sistema preencherá o cadastro com os dados do template automaticamente.
