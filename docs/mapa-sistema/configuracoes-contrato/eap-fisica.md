# EAP Física

A **EAP Física** é o cadastro que armazenará as atividades que suportarão o avanço físico dos contratos. Os itens da EAP serão vinculados às [**Equipes do Contrato**](equipes) e aos [**Itens de Preço e Medição do Contrato**](itens-preco-medicao).​

## Preenchimento manual

Para criar um novo item de EAP, clique no botão <IIcon icon="mdi:plus" height="24" />**Adicionar** na barra de ferramentas acima da tabela.

:::tip[Dica]

Na coluna _Ações_ da tabela, há outras opções para criar itens de EAP:

- Utilize o botão <IIcon icon="ic:baseline-file-copy" height="24" /> para **Duplicar**, i.e., criar um novo item no mesmo nível com uma cópia dos dados do item escolhido.
- Utilize o botão <IIcon icon="ic:baseline-add-circle" height="24" /> para **Adicionar Subitem**, i.e., criar um novo item dentro do item escolhido.

:::

Preencha todos os campos obrigatórios, incluindo o campo _Código_, separando os níveis por pontos. A numeração correta é essencial para garantir o posicionamento do item na hierarquia da EAP. O campo _Nível_ será preenchido automaticamente, sendo 0 o nível raiz.

## Preenchimento via importação de template

Alternativamente, como forma de agilizar a inserção de dados em EAPs complexas, pode-se fazer o download de um template para ser preenchido em aplicativos de planilha eletrônica (ex.: Microsoft Excel) clicando no botão <IIcon icon="ic:baseline-cloud-download" height="24" />**Template**.

Depois que o template for preenchido, basta importá-lo de volta para o Sigo clicando em <IIcon icon="ic:baseline-cloud-upload" height="24" />**Importar**. O sistema preencherá o cadastro com os dados do template automaticamente.
