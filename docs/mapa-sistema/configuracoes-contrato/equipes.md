# Equipes

As **Equipes** são utilizadas para registro do [**Avanço Físico do RDO**](/mapa-sistema/rdo/avanco-fisico).

Uma equipe agrupa os recursos utilizados, ou seja, as [**Mãos de Obra**](/mapa-sistema/configuracoes-sistema/informacoes-rdo-rdf/mao-obra) e [**Equipamentos**](/mapa-sistema/configuracoes-sistema/informacoes-rdo-rdf/equipamentos), além das [**Atividades**](eap-fisica) pelas quais o time será responsável.

:::info[Observação]

Os requisitos mínimos para cadastrar uma nova equipe são uma mão de obra e uma atividade.

Equipamentos são opcionais.

:::
