# Condicionantes Ambientais

O cadastro de **Condicionantes Ambientais** foi criado para controlar as pendências dos empreendimentos em relação ao licenciamento ambiental.​

- Itens em <span style={{color:`green`}}>**verde**</span> ainda não alcaçaram os _Dias para Notificação_ anteriores à _Data para Atendimento_.
- Itens em <span style={{color:`goldenrod`}}>**amarelo**</span> estão dentro dos _Dias para Notificação_ antes da _Data para Atendimento_.
- Itens em <span style={{color:`firebrick`}}>**vermelho**</span> ultrapassaram a _Data para Atendimento_, ou seja, estão atrasados.

## Anexar evidências sobre andamento e/ou cumprimento de condicionantes

Clique no botão <IIcon icon="ic:baseline-edit" height="24" /> na coluna _Ações_ para editar e selecione o botão <IIcon icon="ic:baseline-attach-file" height="24" />**Anexar Documentos** para incluir um documento.

O arquivo será anexado ao empreendimento e estará disponível para consulta entre os [**Documentos do Empreendimento**](documentos).
