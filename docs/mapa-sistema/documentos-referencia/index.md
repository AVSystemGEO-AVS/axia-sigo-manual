# Documentos de Referência

Para acessar os **Documentos de Referência**, clique no <IIcon icon="mdi:menu" height="24" /> **Menu do Usuário** no canto superior direito do cabeçalho (ao lado do nome e hora) e escolha **Documentos de Referência** para acessar.

:::warning[Importante]

Se o menu não apresentar a opção **Documentos de Referência**, significa que o usuário não possui permissão para acessar essa seção.

:::
