# Configurações do Sistema

As **Configurações do Sistema** funcionam como um cadastro de itens gerais que serão utilizados nas outras camadas de funcionalidades do Sigo. Clique no <IIcon icon="mdi:menu" height="24" /> **Menu do Usuário** no canto superior direito do cabeçalho (ao lado do nome e hora) e escolha **Configurações** para acessar.

:::warning[Importante]

Se o menu não apresentar a opção **Configurações**, significa que o usuário não possui permissão para acessar essa seção.

:::

:::tip[Dica]

Recomenda-se preencher os cadastros na ordem apresentada no menu à esquerda (de cima para baixo), pois itens criados em um cadastro acima poderão ser necessários no preenchimento dos cadastros abaixo.

:::
