# Tipos de Componentes

**Tipos de Componentes** são vinculados a [**Componentes do Contrato**](/mapa-sistema/configuracoes-contrato/componentes), que, por sua vez, serão necessários para elaboração dos [**RDOs**](/mapa-sistema/rdo) e [**Coletas Físicas**](/mapa-sistema/coleta-fisica).
