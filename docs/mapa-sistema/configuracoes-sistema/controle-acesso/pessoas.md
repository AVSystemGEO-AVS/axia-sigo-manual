# Pessoas

Este cadastro disponibiliza informações dos colaboradores internos e externos que utilizarão o sistema.

**Pessoas** cadastradas serão vinculadas a **[Usuários](usuarios)**.

:::info[Observação]

Uma pessoa pode possuir múltiplos usuários associados.

Neste caso, todos ficam vinculados ao mesmo e-mail.

:::
