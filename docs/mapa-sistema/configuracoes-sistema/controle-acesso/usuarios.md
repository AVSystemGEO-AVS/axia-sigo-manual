# Usuários

O cadastro de **Usuários** serve para criar e gerenciar as contas utilizadas para acessar o Sigo.

Ao cadastrar um novo usuário, será necessário especificar um _Login_, que será o nome de usuário utilizado juntamente com a senha para logar no sistema. Além disso, deve-se especificar quais _Grupos_ Eletrobras ele poderá acessar na aba [<IIcon icon="mdi:home" height="24" />**Home**](/mapa-sistema/home).

Também será necessário vincular um _Perfil_, que consiste em um agrupamento de permissões que determina quais funcionalidades aquele usuário poderá visualizar e/ou editar.

## Definir permissões customizadas

É possivel customizar as permissões específicas de um usuário, diferenciando-o do perfil escolhido.

Depois que o usuário for criado, clique no botão <IIcon icon="ic:baseline-edit" height="24" /> na coluna _Ações_ para editar e selecione o botão <IIcon icon="ic:baseline-admin-panel-settings" height="24" />**Permissões** para abrir a lista de permissões do sistema.

## Reiniciar autenticação multifator

Clique no botão <IIcon icon="ic:baseline-key" height="24" /> na coluna _Ações_ para reiniciar a autenticação multifator do usuário.

_Disponível apenas para usuários com perfil **Administrador**._
