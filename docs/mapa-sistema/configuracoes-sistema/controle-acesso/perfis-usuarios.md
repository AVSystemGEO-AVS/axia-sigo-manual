# Perfis de Usuários

Os **Perfis de Usuários** permitem definir conjuntos de permissões a serem vinculados aos [**Usuários**](usuarios).

Crie um novo perfil clicando em <IIcon icon="mdi:plus" height="24" />**Adicionar** na barra de ferramentas no topo da tabela.

Após criado, edite clicando no botão <IIcon icon="ic:baseline-edit" height="24" /> na coluna _Ações_ e selecione quais telas do Sigo os usuários com aquele perfil poderão acessar.

Marque apenas a opção _Leitura_ caso deseje que o usuário tenha acesso aos dados em modo somente leitura.

Marque a opção _Escrita_ para permitir que o usuário insira novos itens ou realize alterações nos itens salvos.

Caso nenhuma das opções esteja marcada, o Sigo não permitirá que o usuário visualize aquela funcionalidade.
