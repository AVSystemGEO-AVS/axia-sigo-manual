# Empreendimentos

No Sigo, **Empreendimentos** funcionam como agrupadores de **[Contratos](/mapa-sistema/contrato)**. Ao cadastrar um empreendimento, ele será incluído no **[Menu de Empreendimentos](/mapa-sistema/grupo#menu-de-empreendimentos)** da aba **Grupo** correspondente.
