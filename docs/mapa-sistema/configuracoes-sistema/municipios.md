# Municípios

**Municípios** são vinculados a diversos outros cadastros, como [**Empresas**](empresas), [**Instalações**](instalacoes) e [**Municípios do Contrato**](/mapa-sistema/configuracoes-contrato/municipios).

Por padrão, todos os municípios do Brasil já estão cadastrados, mas é possível complementar cada item com _CEP_.
