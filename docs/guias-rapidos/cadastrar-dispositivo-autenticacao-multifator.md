# Cadastrar dispositivo para autenticação multifator

:::warning[Importante]

Por motivos de segurança e proteção de dados confidenciais, a Eletrobras exige que os usuários do Sigo utilizem autenticação de duas etapas para acessar o sistema.

:::

:::info[Observação]

Para utilizar esta funcionalidade, será necessário um smartphone com um aplicativo _Authenticator_ (ex.: Google Authenticator, Microsoft Authenticator) instalado.

:::

1. Na tela de **Login**, preencha os campos com seu _Login_ e _Senha_ e clique em <IIcon icon="ic:baseline-login" height="24" />**Entrar**.

2. Na primeira vez que o login for efetuado (ou caso tenha sido solicitado o recadastro para um novo dispositivo), o Sigo apresentará um QR Code.

3. No seu smartphone, abra o app _Authenticator_ de sua preferência e escaneie o QR Code mostrado na tela.

4. Digite o código de uso único de 6 dígitos fornecido pelo app e clique em <IIcon icon="ic:baseline-key" height="24" />**Confirmar**.

:::info[Observação]

Nos acessos seguintes, não será mais necessário escanear um QR Code, apenas digitar o código gerado no aplicativo.

:::

:::warning[Importante]

Caso seja necessário refazer o cadastro do _Authenticator_ para sua conta, entre em contato com um Administrador e solicite o reinício.

:::
