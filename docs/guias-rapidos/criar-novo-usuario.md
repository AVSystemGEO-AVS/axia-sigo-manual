# Criar novo usuário

:::info[Antes de Começar]

Para criar um novo usuário no Sigo, será necessário providenciar as seguintes informações:

- Nome Completo \*
- CPF
- Empresa \*
- Órgão \*
- Matrícula
- Telefone
- Celular
- E-mail \*

Campos marcados com **\*** são obrigatórios.

:::

## Cadastrar pessoa

1. Clique no <IIcon icon="mdi:menu" height="24" /> **Menu do Usuário** no canto superior direito e escolha **Configurações**.

2. No menu lateral, clique em **Controle de Acesso** e depois em <IIcon icon="ic:person" height="24" />**Pessoas**.

3. Na barra de ferramentas acima da tabela, clique no botão <IIcon icon="mdi:plus" height="24" />**Adicionar**.

4. Preencha todos os campos obrigatórios e clique em **Salvar**.

## Cadastrar usuário

1. Após se certificar de que o colaborador está cadastrado em **Pessoas**, clique no menu lateral em **Controle de Acesso** e então em <IIcon icon="ic:baseline-account-circle" height="24" />**Usuários**.

2. Na barra de ferramentas acima da tabela, clique no botão <IIcon icon="mdi:plus" height="24" />**Adicionar**.

3. Digite o nome da pessoa recém-cadastrada para selecioná-la.

:::tip[Dica]

Uma pessoa pode possuir múltiplos usuários associados.

Neste caso, todos ficam vinculados ao mesmo e-mail.

:::

4. Preencha todos os campos obrigatórios, incluindo *Login*, que será o nome utilizado para logar no sistema, e *Perfil*, que atribuirá um conjunto de permissões predefinidas que determinam quais funcionalidades do Sigo o usuário poderá acessar.

5. Clique em **Salvar**.

6. Um e-mail será automaticamente enviado para o endereço cadastrado com um link através do qual o novo usuário poderá criar uma senha.
