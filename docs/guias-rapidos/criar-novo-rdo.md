# Criar novo RDO

:::info[Antes de começar]

Antes de criar um RDO, certifique-se de que todos os dados do **[contrato](/mapa-sistema/configuracoes-contrato)** foram devidamente preenchidos.

Para adicionar um RDO a uma **[obra](/mapa-sistema/configuracoes-contrato/obras)**, é necessário que a mesma possua os campos _Data de Início da Execução_ e _Data de Término da Execução_ já preenchidos.

:::

## Cadastrar novo RDO

1. Na aba <IIcon icon="mdi:home" height="24" />**Home**, selecione o grupo, i.e., a concessionária responsável pelo contrato no qual foi cadastrada a obra que receberá o novo RDO.

2. Na aba do grupo escolhido, no menu à esquerda, localize o empreendimento no qual o contrato foi criado e abra a sublista de contrato clicando em <IIcon icon="ic:baseline-chevron-right" height="24" />.

3. Clique duas vezes no contrato.

4. Na aba de <IIcon icon="ic:baseline-request-quote" height="24" />**Contrato**, no menu lateral, selecione <IIcon icon="ic:baseline-summarize" height="24" />**RDOs**.

5. Na barra de ferramentas acima da tabela, clique no botão <IIcon icon="mdi:plus" height="24" />**Adicionar**.

6. Selecione a _Obra_ na lista. O campo _Data_ será automaticamente preenchido com a próxima data disponível para criação de RDOs para a obra escolhida.

:::info[Observação]

Não é possível escolher a data do RDO a ser criado.

Caso seja necessário avançar até uma data futura, crie todos os RDOs até a data necessária.

:::

7. Caso não tenha acontecido trabalho no dia, marque a opção _Não houve atividade na data selecionada_ e informe a _Justificativa_.

8. Clique em **Salvar**.

9. O usuário será automaticamente redirecionado para a aba <IIcon icon="ic:baseline-summarize" height="24" />**RDO**.

## Preencher dados básicos do RDO

1. Antes de poder utilizar todas as funcionalidades do RDO, como registrar [**Avanço Físico**](/mapa-sistema/rdo/avanco-fisico), anexar documentos e gerar PDFs, é necessário completar informações básicas.

2. No topo da tabela de **Condições Climáticas por Município**, clique no botão <IIcon icon="mdi:plus" height="24" />**Adicionar**.

3. Selecione um _Município_ e preencha todos os dados sobre o clima no dia e local.

4. Clique em **Incluir** para adicionar o item à lista e repita o processo para todos os municípios a serem reportados.

5. Clique em **Salvar**.

6. A partir de agora, todas funcionalidades do RDO estarão disponíveis (a menos que o campo _Não houve atividade na data selecionada_ tenha sido marcado).
