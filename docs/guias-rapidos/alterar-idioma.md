# Alterar idioma

1. Clique no <IIcon icon="mdi:menu" height="24" /> **Menu do Usuário** no canto superior direito e clique em uma das três bandeiras apresentadas. O Sigo possui as opções de **Português**, **Inglês** e **Espanhol** para idioma da interface, respectivamente.

:::info[Observação]

Esta configuração altera apenas o idioma da interface do sistema, sem modificar o conteúdo de nenhum item cadastrado.

:::
